使用指南

Windows
  1. 下载并运行 LongMarchServer.exe
  2. 随后运行 LongMarchClient.exe
  3. 若被 SmartScreen 拦截，选择“仍要运行”
Linux
  1. 下载 Long_March.tar 并解压
  2. 进入解压目录后依次执行：
$ python server.py
$ python client.py

如有 Python3，请将命令中的 python 替换为 python3。
